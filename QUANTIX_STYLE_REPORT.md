## 🛠️ 技术实现

### 1. 颜色系统升级

**TailwindCSS 配置** (`index.html`):
```javascript
colors: {
  quantix: {
    black: '#050505',        // 主背景
    purple: '#2E5CFF',       // 主强调色
    charcoal: '#1A1A1C',     // 用户气泡
    border: '#1a1f2e',       // 边框色
    auroraStart: '#8b5cf6',  // 极光起始色
    auroraEnd: '#06b6d4',   // 极光结束色
  }
}
```

### 2. 极光背景效果

**核心实现**:
```css
.quantix-aurora {
  background: radial-gradient(
    circle,
    rgba(139, 92, 246, 0.15) 0%,
    rgba(46, 92, 255, 0.1) 30%,
    rgba(6, 182, 212, 0.08) 60%,
    transparent 70%
  );
  filter: blur(40px);
  animation: auroraFloat 15s ease-in-out infinite;
}
```

**动画特点**:
- 15秒缓慢浮动周期
- 3层极光叠加效果
- 透明度动态变化
- 模糊滤镜增强氛围

### 3. 金属按钮动画

**流光效果**:
```css
.quantix-metal-button::before {
  content: '';
  position: absolute;
  top: 0;
  left: -100%;
  width: 100%;
  height: 100%;
  background: linear-gradient(
    90deg,
    transparent,
    rgba(255, 255, 255, 0.3),
    transparent
  );
  animation: shimmer 2s infinite;
}
```

**特点**:
- 2秒流光周期
- 悬停时向上移动2px
- 增强阴影和发光效果
- 流畅的过渡动画

### 4. 炭黑用户气泡

**样式定义**:
```css
.quantix-user-bubble {
  background: linear-gradient(135deg, #1A1A1C 0%, #0f0f10 100%);
  border: 1px solid rgba(255, 255, 255, 0.08);
  box-shadow:
    0 8px 32px rgba(0, 0, 0, 0.5),
    inset 0 1px 0 rgba(255, 255, 255, 0.05);
}
```
